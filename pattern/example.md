### Как получить загранпаспорт?

В поисках того, кто согласится сделать для Луми новое крыло, мы забрели в "район механиков". Посетив один из магазинчиков, к нам обратился мужчина с предложением помочь. Вот чего он хотел:  
*"Друзья, у меня возникла некоторая проблема, но если вы разрешите ее, я помогу вам.  Дело в том, что мне нужно получить загранпаспорт. Только вот я уже совсем стар стал, и сам не могу разобраться с этими новыми технологиями. Помогите мне, в долгу не останусь."*

![](images/bar.png)

Доступ к порталу **gosuslugi.net**, охраняется суровой цепочкой роутеров. Один из них — легендарный роутер "Батя", настроенный в далеких 90-х. С портала госуслуг нам необходимо получить код активации загранпаспорта нового поколения. Только вот незадача: тебе придется пробить три маршрутизатора, на последнем из которых закрыты все порты, кроме **ICMP**, а сервер госуслуг ответит на ICMP Echo-запрос, **только если в payload будет слово "пж"**. ![](images/dialog_lumi.png)
### Ваша задача:

Вы получаете готовый `docker-compose.yml` с 5 контейнерами:
- `client` — ваша машина;
- `router1` — роутер с IP `10.10.10.1`;
- `router2` — роутер с IP `10.20.20.1`;
- `router_batya` — суровый Linux-роутер, где `iptables` всё блочит, кроме ICMP;
- `gosuslugi` — конечная цель с IP `10.30.30.10`, отвечает на ICMP с payload `"пж"`.

1. Необходимо настроить маршруты на клиенте и роутерах, чтобы ICMP дошёл до `gosuslugi`.
2. Отправить ICMP Echo-запрос со специальным payload: `пж`.
3. Получить ICMP Echo-reply с флагом: `GOSU-PASS-[...код...]`.
4. Написать скрипт `client/ping_gosu.sh`, который должен:
	- послать ICMP с нужным payload (можно `hping3`, `ping -p`, `scapy`, `python`);
	- распарсить ответ и вывести код из полученного флага в `stdout`.

Файловая структура задания
```
/home/student/task2.1.1/
├── client/
│   └── ping_gosu.sh       # ваш скрипт (решение)
├── router1/
│   └── Dockerfile
├── router2/
│   └── Dockerfile
├── router_batya/
│   └── Dockerfile
├── gosuslugi/
│   └── ping_responder.py
└── docker-compose.yml
```
