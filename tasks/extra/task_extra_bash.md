## Охотник за Терминальной Лихорадкой

Терминальная Лихорадка — не просто вирус. Это цифровой хищник, возникший на стыке экспериментального ИИ и заброшенных военных разработок. Его создавали для кибервойны, но он вышел из-под контроля. 

Вирус эволюционирует в реальном времени, учится на ваших действиях и способен: 
- Переписывать свои сигнатуры прямо во время сканирования;
- Маскировать активность под легитимные процессы;
- Атаковать средства мониторинга;
- Оставлять ложные следы в логах.

### Ваша миссия

Создать инструмент для криминалистического анализа логов до того, как вирус завершит тотальное заражение. 

### Сборка инструментария

Создайте файл скрипта: 
```
bash
log_analyzer.sh 
nano
log_analyzer.sh
```

Дайте права на выполнение: 
```
bash 
chmod +x log_analyzer.sh
```

Аргументы командной строки - используйте $1 для доступа к файлу логов 

Для версии V2 используйте getopts для обработки параметров:
```
bash 
while getopts ":o:p:" opt; do
	case $opt in 
		o) output_file="$OPTARG" ;; 
		p) pattern="$OPTARG" ;; 
	esac 
done
```

### Сигнатуры заражения

Вирус оставляет уникальные отпечатки: 
Критические маркеры (ERROR): 
`error`, `fail`, `critical`, `segfault` — признаки борьбы системы

Тактические маркеры (WARNING): 
`warning`, `deprecat`, `notice` — следы маскировки вируса 

#### Скрипт `log_analyzer.sh` 

Принимает файл логов. Выводит оперативную сводку: 
```
text 
- TOTAL: [число]
- ERROR: [число]
- WARNING: [число]
```
Если файл не существует - вывести `ERROR: File [путь] not found` и завершиться с кодом 1
Если нет аргументов - вывести `USAGE: log_[analyzer.sh] [log_file]` и завершиться с кодом 1 

#### Усиленная версия `log_analyzerV2.sh`

Добавляет параметры: -o — файл для доказательств (по умолчанию quarantine_evidence.log), -p — кастомный паттерн для поиска новых мутаций 

Выводит: 
```
text 
SAVED: [число записей] 
FILE: [полный путь] 
```

Пример операции: 
```
bash 
./log_[analyzerV2.sh] -o forensic.log -p 'segfault|critical' 
/var/log/syslog
```

### Цель операции

Найти достаточно артефактов для разработки антидота. Каждая найденная запись — это ключ к алгоритмам Терминальной Лихорадки. 
Вирус уже знает, что вы его ищете. Он будет сопротивляться и атаковать ваши инструменты. Действуйте быстро — система необратимо меняется с каждой минутой.
